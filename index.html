Task 1: Country happiness patterns + trends

Choloropleth Map + Stacked Bar chart

In task one we use a chloropleth map and a bar chart, linked by the year slider to explore happiness patterns and trends.

World Happiness Task 1 by ampiiere's Workspace

Features

Choloropleth Map

Color is used to encode the numeric value of one of the happiness categories

  • Choose which happiness category the color encodes through the radio buttons
  • Year slider shared with the bar chart to decide which yearly data displayed
  • Hover over countries in the map for a tooltip with accurate attribute value
Stacked Bar chart

Each row represents a country, with the country happiness categories stacked to show the total happiness score.

  • Choose the attribute at the bottom of the bars.This is to allow easier comparision of the variable attribute between countries
  • Year slider shared with the map to decide which yearly data displayed
  • Hover over countries in the map for a tooltip with the percentage the happiness category makes up total happiness score
  • Ascending/Descending order of happiness scores
  • Slider to choose how many country data points displayed, to save screen space

Task 2: Discover Inter-variable relationships

Correlation Plot + Parllel Coordinates

For task 2 we use a correlation matrix and a parallel coordinate graph, linked by a year slider to explore inter-variable relationships.

World happiness- Task 2 by ampiiere's Workspace

Features

Correlation Matrix

Provides a rougher, broader overview of how variables relate to each other.

  • Year slider shared with the parallel coordinates graph to decide which yearly data displayed
Parallel Coordinates

Provides a more fine grained view of how variables relate to each other. Colors in lines represent the value of the selected attribute.

  • Drop down menu allows users to select which attribute the colors in the lines are encoded by.
  • Year slider shared with correlation matrix to decide which yearly data displayed
  • Brushing capabilities, drag your mouse over the axes and highlight certain lines. You can do so over multiple axes.

World happiness data

The World Happiness Report is a landmark survey of the state of global happiness.The report continues to gain global recognition as governments, organizations and civil society increasingly use happiness indicators to inform their policy-making decisions.The happiness scores and rankings use data from the Gallup World Poll. The scores are based on answers to the main life evaluation question asked in the poll. The following columns: GDP per Capita, Family, Life Expectancy, Freedom, Generosity, Trust(Government Corruption) describe the extent to which these factors contribute in evaluating the happiness in each country. The Dystopia Residual metric actually is the Dystopia Happiness Score(1.85) + the Residual value or the unexplained value for each country as stated in the previous answer. If you add all these factors up, you get the happiness score so it might be un-reliable to model them to predict Happiness Scores. (from Kaggle world happiness dataset description)
